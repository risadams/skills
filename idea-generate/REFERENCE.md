# Idea Generation Frameworks

This document provides a reference for structured brainstorming techniques to be used within the `idea-generate` skill.

## 1. First Principles Thinking
Break down a problem into its most fundamental truths and build up from there. Avoid reasoning by analogy.

**Process:**
1. **Identify current assumptions**: List everything you "know" or assume about the topic/problem.
2. **Break down to fundamentals**: Question every assumption until you reach a basic physical or logical truth.
3. **Create new solutions**: Reconstruct the idea from the ground up using only these fundamental truths.

## 2. SCAMPER
A checklist used to spark creativity by modifying existing ideas or objects.

- **S**ubstitute: What can be replaced? (People, materials, processes)
- **C**ombine: What happens if you merge two separate ideas/functions?
- **A**dapt: How can an idea from another domain be applied here?
- **M**odify/Magnify: What can be exaggerated or emphasized to add value?
- **P**ut to another use: Could this serve a different purpose entirely?
- **E**liminate: What happens if you remove a core element? (Simplification)
- **R**everse/Rearrange: What happens if you flip the order or perspective?
